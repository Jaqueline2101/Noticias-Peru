# Portal de Noticias Perú - Frontend Angular

Frontend del portal de noticias desarrollado en Angular 17 con conexión a Supabase.

## Configuración

1. Instalar dependencias:
```bash
npm install
```

2. Configurar variables de entorno:
   - Edita `src/environments/environment.ts`
   - Agrega tu `supabaseKey` (clave anónima de Supabase)

3. Ejecutar en desarrollo:
```bash
npm start
```

4. Compilar para producción:
```bash
npm run build
```

## Estructura del Proyecto

- `src/app/services/` - Servicios (Supabase)
- `src/app/components/` - Componentes reutilizables
- `src/app/pages/` - Páginas principales
- `src/environments/` - Configuración de entornos

## Características

- ✅ Conexión con Supabase
- ✅ Listado de noticias
- ✅ Detalle de noticia
- ✅ Filtrado por categoría
